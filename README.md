# CovNatProd

This repository contains data and files from the manuscript entitled “Elucidating the Covalent Binding of Ottelione A and Withaferin A at the Colchicine Site of β-Tubulin” 

---

## 01_Force_field

Modified version of the Amber14SB force field designed to recognize covalent adducts with withaferin A (WIT) and ottelione A (OTT). It also includes Molecular Dynamics parameter files (MDPs).

---

## 02_Docking

Most favorable binding poses obtained from the ensemble covalent docking study.

---

## 03_MD

Coordinate and topology files used for molecular dynamics (MD) simulations.
